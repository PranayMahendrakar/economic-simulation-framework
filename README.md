# 📊 Integrated Economic Simulation Framework

**Author: Pranay M.**

Comprehensive economic model that can predict emergent properties of policy changes across global markets.

## 🌟 Features

| Module | Description |
|--------|-------------|
| **Policy Analyzer** | Analyze economic policy impacts |
| **Market Simulator** | Simulate market dynamics |
| **Trade Flow Modeler** | Model international trade flows |
| **Monetary Policy Engine** | Analyze monetary policy effects |
| **Fiscal Impact Assessor** | Assess fiscal policy impacts |
| **Labor Market Modeler** | Model labor market dynamics |
| **Financial System Analyzer** | Analyze financial system stability |
| **Emergent Property Detector** | Detect emergent economic properties |
| **Scenario Comparator** | Compare economic scenarios |
| **Economic Dashboard** | View economic simulation dashboard |

## 🚀 Quick Start

```bash
ollama pull llama3.2 && ollama serve
pip install -r requirements.txt
python main.py
```
